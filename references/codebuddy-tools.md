# orch CodeBuddy 工具映射

| Claude Code | CodeBuddy 等价 | 说明 |
|-------------|---------------|------|
| Skill("orch:...") | Instruction 文件 | 通过指令系统加载 |
| Agent(subagent_type=...) | Agent 系统 | CodeBuddy Pro 功能 |
| Bash | 终端 | 内置功能 |
| Write/Edit | 编辑文件 | 内置功能 |
| Grep | 文件搜索 | 内置功能 |

## 配置

通过 `.codebuddy/install.js` 安装指令文件到 CodeBuddy 配置目录。
