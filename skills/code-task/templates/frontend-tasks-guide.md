# 前端基础设施任务指南

## 全栈模式：接口契约前置

前端开发依赖接口契约，必须在后端接口确认后才能开始：

```
数据库设计 → 接口契约确认 → 前端任务（Mock开发） → 联调
```

## 常见前端任务类型

| 任务 | 描述 | 交付物 |
|------|------|--------|
| **组件开发** | UI 组件实现 | `src/components/*.vue/tsx` |
| **页面组装** | 页面级组件 | `src/pages/*.vue/tsx` |
| **API 集成** | 接口调用封装 | `src/api/*.ts` |
| **状态管理** | 全局状态 | `src/store/*.ts` |
| **路由配置** | 页面导航 | `src/router/*.ts` |
| **E2E 测试** | 浏览器测试 | `tests/*.e2e.test.ts` |
