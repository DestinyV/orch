# 后端基础设施任务指南

## 全栈模式：数据库与接口契约优先

后端任务执行顺序：

```
数据库设计 → 迁移脚本 → 领域模型 → 仓储层 → 服务层 → API层 → 集成测试
```

## 常见后端任务类型

| 任务 | 描述 | 交付物 |
|------|------|--------|
| **数据库设计** | 表结构设计 | `sql-ddl.md` |
| **迁移脚本** | DDL/DML 脚本 | `migrations/*.sql` |
| **领域模型** | 实体/值对象 | `src/domain/*.ts` |
| **仓储层** | 数据访问抽象 | `src/repository/*.ts` |
| **服务层** | 业务逻辑 | `src/service/*.ts` |
| **API层** | HTTP 端点 | `src/controller/*.ts` |
| **集成测试** | 多模块协作测试 | `tests/integration/*.test.ts` |
